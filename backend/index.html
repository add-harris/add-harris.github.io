# Kanye REST API 🔥

Voila! ... Welcome to the Kanye REST API (Replica). A replica of the Kayne REST project (opens new window).

A free REST API for random Kanye West quotes (Kanye as a Service)

Just to be clear, this is replica of an API I made for returning randomized Kanye Quest quotes. What more could you ever want from an API?

# Usage 📔

# Example Request

curl -v https://quotes-api-sv3nhcltwa-nw.a.run.app/v1/quotes/kanye

This is a live API, freely available on the internet at https://quotes-api-sv3nhcltwa-nw.a.run.app (opens new window)

The source code, along with examples of how to use it, are available here in the project repo (opens new window).

It is also documented with OpenAPI 3 specifications with a Swagger-UI available here (opens new window).

# Tech Stack 👨‍💻

Although the majority of professional experience is with Java 11 and Spring-Boot, this API is actually built using some alternative latest JVM technologies:

  • Java 17 (opens new window)
  • Kotlin (opens new window)
  • Quarkus (opens new window)
  • Maven (opens new window)
  • and Docker (opens new window)

Comprehensively tested with:

  • Junit 5 (opens new window)
  • mockk (opens new window) (a dedicated mocking library for Kotlin)
  • and RestAssured (opens new window)

And has a fully automated CI/CD pipeline in place using:

  • Github (opens new window)
  • Google Cloud Build (opens new window)
  • Google Container Registry (opens new window)
  • Google Cloud Run (opens new window)
