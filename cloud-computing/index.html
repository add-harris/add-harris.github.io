            # "That's lovely, but why?" you ask...

            All the media content for this site is stored and served using Google Cloud Storage (opens new window) (Googles equivalent of Amazon S3).

            They are set up in my personal GCP account and made available publicly using urls like this:

            • https://storage.googleapis.com/portfolio_images_src/portfolio/st-lucia/IMG-5580.JPG (opens new window)

            In fact most of the projects in this portfolio incorporate some form of Google Cloud technology.

            The Kanye REST (replica) API for example uses:

            • Google Cloud Build (opens new window) for CI / CD
            • is stored as a Docker image in Google Container Registry (opens new window)
            • and is deployed and hosted to Google Cloud Run (opens new window).
